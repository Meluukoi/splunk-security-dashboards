# Splunk Security Dashboards

Security monitoring dashboards built in Splunk to visualize authentication activity, process execution, DNS queries, network connections, and account management events.

---

## Overview

This project focuses on building security monitoring dashboards using Splunk Enterprise and Windows telemetry collected through Sysmon and Windows Security Logs.

The objective is to provide analysts with visualizations that improve visibility into authentication events, process execution, network activity, DNS activity, and account management operations.

---

## Lab Environment

### Tools Used

- Splunk Enterprise
- Splunk Universal Forwarder
- Sysmon
- Windows Event Viewer

### Data Sources

- Microsoft-Windows-Sysmon/Operational
- Windows Security Logs
- Windows System Logs
